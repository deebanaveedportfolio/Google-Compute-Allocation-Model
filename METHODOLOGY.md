# Methodology

The project combines economic reasoning with a lightweight Python simulation.

The process consisted of:

1. Identifying a strategic resource-allocation problem.
2. Reviewing publicly available information on AI infrastructure and cloud services.
3. Developing an economic framework based on opportunity cost and shadow pricing.
4. Implementing a simple simulation to pressure-test the framework.
5. Summarizing the findings in a strategy memo.

The simulation is intended as an illustrative decision-support tool rather than a predictive model.
